# 下拉面板 / Dropdown

向下弹出的列表，用于归纳同类型的项目。

## 基础样式

下拉由下拉项目组成。下拉项目内容可以自定义，可以含有图标，文字和展开按钮，或者为纯文字。
多个层级的下拉见级联选择 Cascader。
